## What I do

1. List files in `~/Downloads/screenshots/`
2. Pick the one with the most recent modification time
3. Read it as an image attachment for visual context

## When to use me

Use this when the user says something like "look at the latest screenshot", "check my last capture", or "see what I have in screenshots".
